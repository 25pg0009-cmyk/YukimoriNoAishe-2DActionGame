# YukimoriNoAishe-2DActionGame
雪森のアイシェ～封炎の誓い～
2Dアクションゲーム
## 概要
攻撃とブロックを使い分けステージを攻略し、最終的にボスを倒すゲームです。

- Excel

## 操作方法
## [アウトゲーム時]
- W,A,S,D/十字キーカーソル移動
- Enter/Aボタン決定
- ESC/Bボタン戻る
- L+Aキーステージ全開放
## [インゲーム時]
- A,D/十字キー/Lスティック移動
- Space/Aボタン ジャンプ
- J/Xボタンアイススピア（攻撃）
- K/RBボタンアイスブロック
- Uキー当たり判定表示
